## Description
Frag webapp server application

## Getting started
1. Install npm packages ```$ npm install```

2. Rename .env.example to .env
3. Fill your PostgreSQL database credentials and random string to JWT_KEY

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
